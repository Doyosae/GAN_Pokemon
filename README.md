# Introduction  
Pokemon 이미지 800여 장으로 GAN 모델들을 구현해봅니다.  
데이터세트는 Kaggle의 잘 정리된 포켓몬 정사각형 사진을 사용하였습니다.  
희한하게 배경은 제거된 모양이네요.  
- https://www.kaggle.com/kvpratama/pokemon-images-dataset
